# Quant Trading Strategy Project

A quantitative trading strategy built in Python, covering the full pipeline from raw market data to backtested performance results.

## About

Developed by Hir Shah, Finance & MIS student at Rutgers Business School. This project demonstrates my ability to apply financial concepts and Python programming to build and evaluate a systematic trading strategy.

## What It Does

- Downloads real stock data using the Yahoo Finance API
- Engineers technical indicators: SMA, RSI, MACD, and Volatility
- Generates a weighted composite signal blending trend, momentum, and mean reversion
- Backtests the strategy with no look-ahead bias
- Computes performance metrics: Sharpe Ratio, CAGR, and Max Drawdown
- Visualizes the equity curve vs. a buy-and-hold benchmark

## Technologies Used

- Python 3
- yfinance
- pandas
- NumPy
- Matplotlib
- VS Code
- GitHub
